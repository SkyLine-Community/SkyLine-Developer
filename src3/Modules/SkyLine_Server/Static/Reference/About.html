What is SkyLine

SkyLine is a modern interpreted programming language designed to make and improve the programming world for cyber security researchers across the world! SkyLine plans and attempts to create this affect by having and storing a backend developed just for cyber security and mathematics while also targeting features that can be in relation to the cyber security world. In todays world there are languages such as python, ruby, javascript, go etc that have libraries dedicated to CTF's however still fail to create a world for defensive AND offensive cyber securuty experts. With SkyLine's base system it attempts to help cyber security experts by providing an easier path to developing cyber security solutions and technologies. Whether or not that may be a very very simple IPS or a program that needs to access servers.


SkyLine Syntax

Given that SkyLine try's to introduce new ideas to the world of development one thing that matches that is it's " hybrid syntax " which allows you to multiple styles of programming. For example if you are used to a standard C syntax in conditionals like so if (data) { printf()}; SkyLine allows you to stick to that same exact syntax while also allowing you to remove the option for semicolons or paranthesis around the variables used within the conditional. This really does make SkyLine one of the more different languages out there and causes it to stick out. Below is a code snippet which shows a general idea of how a conditional may work in SkyLine

Example code


let Name = "Jef"

cause CheckName = Func(name) {
if (name == "jef") {
println("Hey there jef!")
};
};
