SkyLine - Your First Program
Below you will find a basic program in SkyLine showing you the most basic program in the world Hello World!

HelloWorld.csc

println("Hello World!")

Running this program will show an output with "hello world" as println will print out the variable in this case "Hello World" Of type string and will direct the output to STDOUT. SkyLine has a few ways of indicating a source code file, currently source code files must end in .csc, .skyline, .sl or .SkyLine It is also important to note that while we are here SkyLine also has a statement which is "print". It does the same thing println does but does not create a new line after a variable or statement is outputted
